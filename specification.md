# テニスコート予約管理アプリ 仕様書（ver.0.1）

---

## 1. 概要

本アプリは、複数メンバーでテニスコートの「抽選・確保状況」を共有・管理するための
シンプルでポップなUIを持つWebアプリである。
スマートフォンからもアクセス可能で、メンバー間の状況把握を容易にする。

---

## 2. 開発環境

| 項目           | 内容                                  |
| -------------- | ------------------------------------- |
| 言語           | Python 3.10                           |
| フレームワーク | Streamlit                             |
| デプロイ環境   | Streamlit Cloud                       |
| バージョン管理 | GitHub（Public）                      |
| 利用デバイス   | PC／スマートフォン両対応              |
| 認証           | なし（共有URLで誰でも閲覧・編集可能） |

---

## 3. 利用目的

- けやきネット、都営スポーツ施設などの抽選・予約状況をメンバーで共有
- 誰が、どこの、どの時間帯を担当しているかを一目で把握
- 見た目が楽しく、操作が簡単なUIで管理負荷を軽減する

---

## 4. 管理項目

| 項目名 | 内容                                                 |
| ------ | ---------------------------------------------------- |
| 施設名 | テニスコートなどの名称（例：けやきねっと、都営など） |
| 日付   | 予約・抽選対象日                                     |
| 時間帯 | 例：9:00-11:00、11:00-13:00など                      |
| 担当者 | ニックネームを使用                                   |
| 状態   | 「確保」「抽選中」「中止」などを選択                 |

---

## 7. 運用方法

1. GitHubにソースコードを配置（Public設定）
2. Streamlit Cloudでデプロイ
3. 共有リンクをメンバーに通知
4. スマホまたはPCから直接編集・閲覧
