
# テニスコート予約管理アプリ 仕様書（v1.0）

---

## ■ 概要

本アプリは、複数人で行うテニスコートの「抽選・予約」情報を共有・管理するためのWebアプリケーションである。
主にスマートフォンから操作しやすいUIを想定し、月単位のカレンダーで直感的に予約状況を確認できるようにする。

---

## ■ 目的

- 誰が・どこの施設を・いつ予約／抽選しているかを全員が把握できるようにする
- 抽選の申し込み期間や確保状況を忘れずに管理する
- LINEやスプレッドシートでは管理しにくい情報を、見やすいUIで一元化する

---

## ■ 機能概要

| 項目         | 内容                                                                                  |
| ------------ | ------------------------------------------------------------------------------------- |
| 表示形式     | 月カレンダー表示（スマホ対応）                                                        |
| 表示項目     | 施設名、時間帯（10分単位）、担当者（ニックネーム）、状態（確保・抽選中・中止）        |
| 状態別カラー | 確保＝🟩緑／抽選中＝🟨黄／中止＝⬜灰                                                  |
| 操作         | カレンダー上の日付をタップ／クリックで詳細モーダル表示`<br>`→ 登録・編集・削除可能 |
| フィルタ     | 施設名・担当者で絞り込み表示                                                          |
| 施設入力     | 直接入力（固定選択ではない）                                                          |
| 表示範囲     | 月単位（初期値は当月）                                                                |
| 保存方法     | 初期はCSVファイル、将来的にGoogle Sheets連携も視野                                    |
| 対応デバイス | スマホ・PC両対応（レスポンシブデザイン）                                              |
| 時間指定     | 10分単位（例：09:10〜10:20など自由入力）                                              |
