# プロダクトバックログ

---


## 1. 優先度 ★★★（最優先）

### ● 抽選リマインダー機能（v2.0）

* 毎年 / 毎月 / 毎週 の 3 種類のルールに基づく期間設定
* 期間中はトップ画面上部に常時メッセージ表示
* 固定文言・複数メッセージ対応
* リマインダー設定の追加／削除に対応

### ● 参加履歴管理（v2.1）

* 各イベントの参加表明履歴を保存
* 1名につきコメントは1つ（最新を上書き）
* 表示は最新順（降順）
* 削除は不可
* コメントは改行対応

---

## 2. 優先度 ★★☆

### ● カレンダースワイプ操作（v2.2）

* スマホ環境で左右スワイプにより月切替を可能にする
* デスクトップ環境は従来の矢印ボタン操作を維持

### ● カレンダーからの自動遷移（v2.2）

* 日付選択 → 自動で新規登録画面へ遷移
* イベント選択 → 自動で参加表明画面へ遷移
* 操作ステップを減らし直感的 UI を実現
