# プロダクトバックログ

---

## 1. 概要

本ドキュメントは、テニスコート予約管理アプリの機能や改修案を整理し、優先度順に管理するためのものです。

---

## 2. 機能リスト（優先度順）

| 優先度 | 機能名                       | 内容概要                                   | 担当 | 想定リリース |
| :----: | ---------------------------- | ------------------------------------------ | ---- | ------------ |
| ★★★ | 抽選申し込み期間リマインダー | 締切前に通知（メール or 画面上リマインド） | TBD  | v2.0         |
| ★★☆ | 参加履歴管理                 | 参加表明の履歴保存、修正履歴確認           | TBD  | v2.1         |
| ★★☆ | 抽選結果の自動反映           | 抽選期間終了時に確保／不成立を自動更新     | TBD  | v2.0         |
| ★☆☆ | CSV自動バックアップ          | 定期的に予約データをバックアップ           | TBD  | v2.3         |
| ★☆☆ | 施設別統計表示               | 参加人数・予約状況の集計表示               | TBD  | v2.2         |
